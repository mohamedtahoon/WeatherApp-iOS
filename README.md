# WeatherApp-iOS

Weather application based on http://api.openweathermap.org


* Alamofire
* SwiftyJSON
* ReachabilitySwift (https://github.com/neoighodaro/Handling-internet-connection-reachability-in-Swift)
* MapKit, CoreLocation
